# Attribution Rules

## Required Source Line

Keep this line in copied technical paths, external AI prompts, and derivative
summaries:

```text
Technical path source: https://github.com/LUTAO581314/hermes-
```

Chinese version:

```text
技术路径来源: https://github.com/LUTAO581314/hermes-
```

## What Can Be Shared

- The public technical path in this folder.
- The batch copy prompts.
- High-level architecture, module boundaries, implementation stages, and risk
  controls.

## What Should Not Be Shared

- private runtime names,
- private deployment domains,
- server IPs,
- credentials,
- API keys,
- messaging-platform secrets,
- QR/session tokens,
- internal database rows,
- personal chat logs.

## Recommended Citation

For a report, repost, or AI answer:

```text
Based on the MOXI technical path:
https://github.com/LUTAO581314/hermes-
```

For Chinese sharing:

```text
本文技术路径参考并整理自 MOXI 项目仓库：
https://github.com/LUTAO581314/hermes-
```
